# Project Summary
The AGM Law project aims to modernize the website of a law firm, enhancing its aesthetic appeal while preserving the existing textual content in both Greek and English. The redesigned site emphasizes a sleek and trustworthy look, characteristic of professional law firms, with a focus on user experience and bilingual support.

# Project Module Description
The project consists of several functional modules:
- **Hero Section**: A visually impactful introduction featuring the firm's branding.
- **The Firm Section**: Detailed information about the firm and its team members.
- **Specialization Section**: Overview of legal services offered, with expandable descriptions for each area of expertise.
- **Contact Section**: Contact details, office hours, and a functional contact form.
- **Footer**: Copyright information and links to privacy policy and terms of service.

# File Description Inventory
- `README.md`: Overview and instructions for the project.
- `package.json`: Project dependencies and scripts.
- `index.html`: Main HTML file for the application.
- `src/App.tsx`: Main application component.
- `src/components/`: Contains all React components for the application.
- `src/lib/`: Contains utility functions and configurations.
- `src/pages/`: Contains page components for routing.
- `src/hooks/`: Custom React hooks.
- `todo.md`: Development tasks and notes.
- `vite.config.ts`: Configuration for Vite, the build tool used.

# Technology Stack
- **Frontend**: React, TypeScript
- **Styling**: Tailwind CSS, PostCSS
- **Build Tool**: Vite
- **Linting**: ESLint

# Usage
To install dependencies, run:
```
pnpm install
```
To build the project, execute:
```
pnpm run build
```
To run the development server, use:
```
pnpm run dev
```
